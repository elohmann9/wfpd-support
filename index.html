IT & Radio Quick Reference Console

Locate symptoms below to inspect recommended actions and escalation triggers.

Steady Green LED

BK M150
LED is solid green, screen is completely blank, and controls do not respond.
Correction Action:
  • The transceiver is locked in boot. Needs firmware reflash.
  • Plug programming cable into the transceiver body (not head).
  • Critical: Flash firmware to transceiver AND both control heads. Flashing only one will fail.
Escalate: Reflash requires binary files.

CHID # ERROR

APX 8500
Radio screen locks out displaying "CHID # ERROR".
Correction Action:
  • Check programming cable—must connect to the receiver body, never the control head mic port.
  • Verify the head-to-body link cable is fully seated on both ends.
  • New units showing this likely have a blank config codeplug.
Escalate: Required if connections are OK.

Extra CH on Display

APX 8500
Dual-head unit shows "Extra CH" or screen sync issues.
Correction Action:
  • Dual-head configuration must be active in CPS R38 profile before writing to the radio.
  • If broken, change setting in offline CPS profile first, then re-write the radio.
Escalate: All writes require sign-off.

Starlink Offline

Starlink
Rig internet is down; cell coverage is poor and Starlink status is offline in NCM.
Correction Action:
  • Confirm Starlink router power is active (check vehicle inverter).
  • Verify Bypass Mode is toggled on in the Starlink App.
  • Check ethernet adapter seats in both router and Cradlepoint WAN port.
Escalate: If hardware has power but fails.

CAD Tablet Offline

iPad / WiFi
Tablet Command iPad does not update dispatch logs.
Correction Action:
  • Verify iPad is on the primary network: WFPD-[UNIT].
  • Avoid the -GUEST network, as guest rules block CAD sync routing.
  • Renew lease. The IP must resolve to 10.7.X.50 (static DHCP).
Escalate: If IP is correct but CAD fails.

NCM Showing Offline

NetCloud
All district apparatus show offline in NCM simultaneously.
Correction Action:
  • Visit status.cradlepoint.com to check system status.
  • Cradlepoint cloud portal maintenance regularly flags all routers offline even when vehicle internet is working fine.
Notice: Do not reset hardware. Wait.
